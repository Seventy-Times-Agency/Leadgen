# Leadgen Bot — Концепция продукта

## 1) Для кого

- Таргетологи
- SMM-специалисты
- Маркетинговые агентства
- Фрилансеры, которые продают digital-услуги бизнесу

## 2) Что делает бот

Пользователь вводит нишу и регион (например, "стоматологии, Москва"),
бот собирает до 50 компаний и по каждой формирует мини-досье:

- контакты (телефон, сайт, адрес)
- соцсети (если найдены)
- рейтинг и отзывы
- AI-оценка лида (или fallback-оценка, если нет ключа Anthropic)
- рекомендации: как зайти в диалог, какие боли закрывать

## 3) Ценность

Вместо ручного поиска и анализа бот за минуты отдаёт готовую базу,
пригодную для прозвона, сообщений и подготовки коммерческого предложения.

## 4) Пользовательский сценарий

1. `/start`
2. Ввод ниши
3. Ввод региона
4. Подтверждение поиска
5. Получение:
   - аналитической сводки по базе
   - топа лидов с карточками
   - Excel-файла с полной выдачей

## 5) Два режима качества

- **Full AI mode**: при наличии `ANTHROPIC_API_KEY`
- **Fallback mode**: при отсутствии ключа — эвристическая оценка

Таким образом, бот не "падает" в ожидании ключей, а остаётся рабочим.

## 6) Где запускается

- Код хранится в GitHub
- Деплой на Railway
- Все чувствительные настройки задаются через Railway Variables
